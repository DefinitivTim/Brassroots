# Changelog

All notable changes to **Brassroots** will be documented in this file.

The format is loosely based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the pack roughly follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html) (`MAJOR.MINOR.PATCH`):

- **MAJOR** — world-breaking changes (worldgen, biome, or structure-mod changes that affect existing saves)
- **MINOR** — new content / mod additions that are save-safe
- **PATCH** — config tweaks, performance fixes, save-safe mod updates
