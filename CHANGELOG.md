# Brassroots

## v1.0.0-alpha1

Welcome to the very first playable build of Brassroots. It runs on Minecraft 1.21.1 with NeoForge. This is where the journey starts, so everything in this list is brand new.

Brassroots is built around three ideas. You build with Create. You take to the skies with Create: Aeronautics. And you head out to explore a world that is worth exploring. Trains, logistics, and the rest of the Create family are along for the ride as addons.

This release brings 379 mods, the built in Brassroots Essentials resource pack, and Complementary shaders with Euphoria Patches.

## Brassroots Essentials

Brassroots Essentials is a resource pack that ships inside the pack and loads on its own through Paxi. It gives the menus their own look instead of leaning on vanilla art.

For this release it restyles the core interface widgets. That means custom button textures, including the normal, disabled and highlighted states, and custom slider textures for both the track and the handle. Each one comes with its own scaling data so the buttons and sliders stay crisp at any size.

---

## Configuration Notes

Paxi loads the Brassroots Essentials resource pack for you, along with the global datapack and resourcepack folders, which are empty for now. KubeJS is installed with starter client, server and startup scripts ready for future recipes and tweaks. Biolith ships with a default config for biome generation.

Brassroots is an alpha. Expect rough edges, balance changes, and a mod list that will keep shifting as the pack grows. Thanks for playing this early.
