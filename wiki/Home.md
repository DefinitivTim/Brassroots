# Brassroots Wiki

Welcome to the **Brassroots** wiki — the documentation hub for the modpack.

Brassroots is a cozy, Create-focused Minecraft 1.21.1 / NeoForge pack with ~370 mods. The pack mixes deep Create automation with cozy survival, building, cooking, train travel, and rich worldgen.

## About the name

**Brassroots** = **"brass"** (Create's signature material — cogs, brass casings, the whole automation feel) + **"grassroots"** (community-driven, down-to-earth, bottom-up). The name signals what the pack is about: Create at the core, but cozy and approachable rather than min-max grindy.

## Contributing to the wiki

This wiki lives in a separate git repository (`Brassroots.wiki.git`) attached to the main repo. To edit:
